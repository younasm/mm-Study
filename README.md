## 📌 Overview

This repository is the official companion to the research paper:

> **"mm-Study: Activity Recognition in Study Environments Using mmWave Radar Micro-Doppler Signatures Feature Fusion in Tabletop Scenarios"**

This work investigates the use of millimeter-wave (mmWave) radar for non-intrusive human activity recognition (HAR) in study environments. By leveraging micro-Doppler signatures and feature fusion strategies in tabletop scenarios, the proposed framework aims to enable privacy-preserving, contactless monitoring of study-related activities.
